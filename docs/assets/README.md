# Banner Image Guide

This directory contains visual assets for the README.

## Current Banner

The banner image (`banner.png`) should be a professional, wide-format image (approximately 1200x400 pixels) that visually represents:

### Design Concept
- **Left side**: Abstract representation of conversation/chat bubbles or text fragments
- **Middle**: Transformation process (funnel or pipeline visualization) with semantic atoms (geometric shapes like diamonds/cubes)
- **Right side**: Organized, optimized context flowing into a new session
- **Color scheme**: Professional blues, teals, and whites with subtle gradients
- **Style**: Clean, modern, minimal - suitable for enterprise/developer audience

### Recommended Tools to Create
1. **Figma** (Free): Use geometric shapes and gradients
2. **Canva** (Free tier): Search for "tech banner" templates
3. **Excalidraw** (Free): Hand-drawn technical style
4. **PowerPoint/Keynote**: Create shapes and export as PNG

### Alternative: ASCII Banner (Temporary)

If you don't have a graphic yet, you can temporarily use an ASCII art banner in the README:

```text
┌───────────────────────────────────────────────────────────────────┐
│                    CONTEXT HANDOVER                               │
│                                                                   │
│   [Chat] ──→ [🧠 Extract] ──→ [⚡ Optimize] ──→ [🎯 Handover]    │
│                                                                   │
│   Preserve Semantic Continuity Across LLM Session Boundaries     │
└───────────────────────────────────────────────────────────────────┘
```

### Image Specifications
- **Format**: PNG (supports transparency)
- **Dimensions**: 1200x400 pixels (3:1 aspect ratio)
- **File size**: Keep under 500KB for fast loading
- **Alt text**: "Context Handover workflow diagram showing semantic extraction and optimization"

### Where to Place
Once created, save your banner image as `banner.png` in this directory (`docs/assets/`).

The README already references it at: `![Context Handover Banner](docs/assets/banner.png)`
